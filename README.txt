------------- BEFORE RUNNING:
To run, the following text files need to be on source folder of the project:
       - negData.txt 
       - posData.txt
       - negWords.txt
       - posWords.txt
       - stopWords.txt

------------- REFERENCES:
posData and negData reviews extracted from:
       https://github.com/dennybritz/cnn-text-classification-tf/tree/master/data/rt-polaritydata

negWords and posWords files extracted from:
Minqing Hu and Bing Liu. "Mining and Summarizing Customer Reviews." 
       Proceedings of the ACM SIGKDD International Conference on Knowledge 
       Discovery and Data Mining (KDD-2004), Aug 22-25, 2004, Seattle, 
       Washington, USA, 
       https://gist.github.com/mkulakowski2/4289441
       https://gist.github.com/mkulakowski2/4289437

Stop words extracted from:
    https://www.ranks.nl/stopwords
